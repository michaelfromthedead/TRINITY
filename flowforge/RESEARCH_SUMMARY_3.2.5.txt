================================================================================
KEY FACTS
================================================================================

1. LiteGraph Version: Custom fork from ComfyUI (FlowForge-maintained)
   Location: /src/litegraph/src/LGraphCanvas.ts (8,639 lines)

2. Dual Canvas Architecture:
   - Primary canvas: main interactive work area
   - Background canvas: groups, connections, static elements
   - Both render via hook system (no monolithic rendering)

3. Built-In Features (Existing):
   ✓ Zoom/Pan (mouse wheel, middle-drag)
   ✓ Fit to view (auto-zoom to bounding box)
   ✓ Viewport tracking (knows what's visible)
   ✓ Grid snapping
   ✓ Node groups and frames
   ✓ Execution order visualization

4. Missing Features:
   ✗ Minimap/Overview navigator
   ✗ Timeline/History view
   ✗ Breadcrumb navigation
   ✗ In-canvas orientation indicator

================================================================================
TECHNICAL FOUNDATION
================================================================================

Rendering Hooks Available (Perfect for Extension):
- onDrawBackground(ctx, area) - Behind all elements
- onDrawForeground(ctx, area) - Above nodes
- onDrawOverlay(ctx) - UI elements [BEST FOR MINIMAP]
- onRender(canvas, ctx) - Full render callback
- onRenderBackground(canvas, ctx) - Alternative background

Viewport Data Available:
- canvas.ds.visible_area - Current viewport rectangle [x, y, w, h]
- canvas.ds.offset - Pan offset [x, y]
- canvas.ds.scale - Current zoom level
- canvas.graph.nodes - All nodes in graph
- canvas.graph.getBoundingBox() - Full graph extent

Data Access Pattern:
const canvas = graphCanvasRef.value
const nodes = canvas.graph.nodes
const viewportRect = canvas.ds.visible_area
const zoomLevel = canvas.ds.scale
const panOffset = canvas.ds.offset

================================================================================
CRITICAL CODE PATTERNS
================================================================================

Coordinate Transformation (Canvas to Graph):
  graphX = (canvasX + offset[0] * scale) / scale
  graphY = (canvasY + offset[1] * scale) / scale

Graph to Canvas (What User Sees):
  canvasX = graphX * scale - offset[0] * scale
  canvasY = graphY * scale - offset[1] * scale

Minimap Rendering Scale:
  scale = min(minimapWidth / graphWidth, minimapHeight / graphHeight)

Panning to Location:
  canvas.ds.offset[0] = targetGraphX - (viewportWidth / 2)
  canvas.ds.offset[1] = targetGraphY - (viewportHeight / 2)
  canvas.setDirty(true, true)  // Force redraw
