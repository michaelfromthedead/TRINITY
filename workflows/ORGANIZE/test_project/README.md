# Spin Physics Research

A research codebase for numerical analysis of spin dynamics in condensed matter systems.
This project combines Python simulation code, Jupyter analysis notebooks, structured
documentation, and raw experimental data outputs.

## Structure

- `src/` — core simulation and analysis code
- `docs/` — project documentation and research notes
- `data/` — datasets and experimental outputs (not committed)
- `tests/` — automated tests for simulation modules

## Requirements

Python 3.11+. See `pyproject.toml` for dependencies.

## Running

```bash
python src/main.py --config config/default.toml
```
